Javascript Calculator

Portfolio #01

Please go to my Github (Linked Below) to view my code! This was a basic project inspired from a video made by "Web Dev Simplified". In this project, I created a Functional Calender made using HTML, CSS, and Javascript. My goals for this project was to self-learn and better understand concepts of Javascript, as well as to connect a script to a UI on my own personal website!

This was my second interaction with JS. I not only got more practice with HTMl and CSS, but to really learn and understand JS step by step. As I have previously learned Java, JS was only really different in syntax, being a scripting langauge, and only being able to run on a browser. I am happy with the results of the project, and the implementation of it into my website. I believe I was able to achieve a better understanding of JS in a more hands-on way.
